# Resumo do Projeto

## Visão Geral
Este projeto consiste em um sistema bancário simples desenvolvido em Python. O sistema permite o cadastro de usuários, gerenciamento de saldo, realização de saques e depósitos.

## Objetivo Principal
Criar uma aplicação que simule operações bancárias básicas, permitindo aos usuários gerenciar suas contas de forma simples e eficiente.

## Público-Alvo
Usuários que necessitam de um sistema simples para gerenciamento financeiro pessoal.

## Valor Estratégico
O sistema serve como uma base para entendimento de conceitos de programação orientada a objetos em Python, gestão de dados e operações financeiras básicas.
